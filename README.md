Лендинг “Сложно сосредоточиться”

Адаптивное приложение с переключением тем оформления.

Реализовала:
1) систему тем через CSS-переменные
2) переключение темы на JavaScript
3) полную адаптивность под разные устройства

Стек: JavaScript, CSS (variables, media queries), HTML

Проект опубликованный на gh-pages: https://dariyajam.github.io/slozhno-sosredotochitsya/
